# 使用 Truffle-VUE Box

这是一个基于vue的solidity开发Dapp框架.
[English Doc](README.md)
## 开箱使用

### Install truffle

> 第一步

```
$ (sudo) npm install -g ethereumjs-testrpc
```

> 第二步

```
$ (sudo) npm install -g truffle
```

> 第三步

```
$ https://github.com/huarxia/truffle-vue.git
$ cd truffle-vue
```

> 第四步

```
$ npm install
```

> 第五步 新开一个shell窗口
 
```
$ testrpc
```
> 第六步 回到刚才的 shell窗口

```
$ truffle compile
$ truffle migration
$ npm run dev
```
### develop shell

> 编译智能合约

```
$ truffle compile
```
> 链接本地区块链

```
$ truffle migration
```
